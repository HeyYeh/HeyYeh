Behind the app:

This is a work in progress project and started as me wanting a simple app that would randomly give a musical scale and actually show what the scale is, unlike a lot of other web apps out there. At the same time, you would need the option of being able to hide the notes or the key signature to help with your memory, and you would need to be able to hear the scale or notes to train your ear too (not currently in this version (sorry!)). I also wanted to customise it based on the grade of the user, because there’s no point flooding beginners with heaps of jargon that they won’t understand or care about at that stage.

What you see above is an early version of that project, and I’m still working on it, but I thought posting it now would be at least somewhat useful to people with even a passing interest in programming because I documented the hell out of this one. When I work with an API for the first time, in this case it’s the amazing Vex.Flow which brilliantly draws sheet music bars to a HTML5 canvas (much better than just a picture for a lot of reasons, but I won’t get into that), I like to document everything I do in comments so that I actually learn. In the early stages, I’m reading other people’s documentation and tutorials and re-writing it definitely helps it to sink in thanks to good ol’ multi-sensory memorisation!

Instead of boring you with stuff though, feel free to trawl through the JavaScript using your browser's developer tools if you’re comfortable with that, OR take a look at the Code Pen snippet of this below!

I should give a couple of disclaimers: first of all, because this is an older version it is not even close to being optimised for smaller screens, purely because I had no idea how to make the HTML5 canvas responsive when I was writing this before. Second, the CodePen snippet below is probably the best way to browse the code, but it will take a few moments to load because it's a long ass snippet! Third, this isn't necessarily the best way to go about making something like this, it is just how I problem solved it and the beauty of programming is that there are often many ways and tools to help you paint a picture. Definitely read around and make your own paintbrush/paint roller/stick with a sponge on the end!
